# Project Brief - Sentra AI & Elite Ecosystem

## Core Requirements & Goals
- Merealisasikan **Sentra Artificial Intelligence** sebagai *Augmented Clinical Intelligence Layer* untuk menyelamatkan nyawa di Indonesia.
- Mengembangkan dan mengoptimalkan model **Gemini** dalam alur kerja engineering kecepatan tinggi.
- Memastikan integritas teknis absolut melalui protokol **Brutal Honesty** dan sistem eksekusi **JET**.

## Project Scope
- Pengembangan infrastruktur **The Abyss** (Cerebrum, API Gateway, Orchestration).
- Implementasi modul klinis: **Sentra Assist Widget**, **Sentra Referralink**, dan **Sentra AADI**.
- Akselerasi portofolio engineering Chief di seluruh ekosistem Google.
